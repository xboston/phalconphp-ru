Stack Overflow

Если нравится и пользуетесь Stack Overflow используйте тег phalcon для поиска и помечания своих вопросов. Пожалуйста, учтите что:
  • Stack Overflow не предназначен для поддержки Phalcon
  • Не дублируйте ошибки. Наблюдайте за трекером ошибок на Github
  • Не оставляйте пожеланий. Используйте трекер ошибок на Github для этого
  • Указывайте тег "phalcon" в вопросах
  • Указывайте что испольуете "Phalcon" в описании вопроса
Есть так же чат в Stack Overflow для неформального общения.
