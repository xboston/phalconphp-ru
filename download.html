Начало работы

Phalcon PHP написан на C и не привязан к какой-либо платформе. В результате он доступен для Microsoft Windows, GNU/Linux, FreeBSD и Mac OS X. Вы можете скачать готовый пакет для своей системы, либо собрать его самостоятельно.

Windows

Для пользователей Windows достаточно скачать DLL библиотеку. Просто добавьте в ваш файл php.ini подключение расширения phalcon: 
extension=php_phalcon.dll
Перезагрузите веб-сервер, готово.

Linux

Phalcon в Linux проще собрать из исходных кодов.

Требования

Понадобятся установленные пакеты
  • Инструменты разработки PHP 5.x
  • Компилятор GCC
#Ubuntu
sudo apt-get install php5-dev php5-mysql gcc

#Suse
yast2 -i php5-pear php5-devel php5-mysql gcc

Компиляция

Для создания расширения из исходного кода на Си выполните шаги: 
git clone git://github.com/phalcon/cphalcon.git
cd cphalcon/build
sudo ./install
И добавьте в файл php.ini: 
extension=phalcon.so
Перезагрузите веб-сервер

Arch Linux

Для ArchLinux существует PKGBUILD

FreeBSD

Для FreeBSD существует порт (port). Устанавливается простыми командами: 
# pkg_add -r phalcon
или 
# cd /usr/ports/www/phalcon && make install clean
